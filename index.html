API de armazenamento local

local storage e session Storage

O objeto localStorage armazena os dados sem data de expiração. Os dados não serão excluídos quando o navegador for fechado.

O objeto sessionStorage armazena os dados para apenas uma sessão. Os dados são deletados quando o usuário fecha a aba específica do navegador.
